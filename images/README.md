# Recipe Images

This folder contains photos of Apa's recipes.

## How to Add Images

1. **Take or upload a photo** of the finished dish
2. **Name the file** to match the recipe ID (e.g., `arroz-con-leche.jpg`)
3. **Add the file** to this folder
4. **Update the recipe** in `recipes.json` to include the image filename:
   ```json
   {
     "id": "arroz-con-leche",
     "title": "Arroz con Leche",
     "image": "arroz-con-leche.jpg",
     ...
   }
   ```

## Supported Formats

- JPG/JPEG
- PNG
- WebP

## Recommended Image Size

- **Width:** 800-1200px
- **Aspect ratio:** 4:3 or 16:9
- **File size:** Keep under 500KB for fast loading

## Tips

- Use natural lighting for food photos
- Show the dish from a slight angle (not straight down)
- Include a simple, clean background
- Resize large images before uploading to keep the site fast
